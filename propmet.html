Prompt-Optimizer

By deploying the extremely lightweight Qwen2.5-1.5B model locally as an "optimizer," this system can instantly transform users' vague and unstructured Chinese requirements into high-information-density professional English commands, thereby saving tokens.

Prompt Structuring

  1. Structured Transformation Breaking down requirements into frameworks to minimize cloud model inference ambiguity
  2. Information Density Although information volume has increased by 20.5x, this is not redundancy; rather, it is the refinement of "vague intent" into "precise logic."

import streamlit as st
from openai import OpenAI
import time

st.set_page_config(page_title="Prompt Assistant(qwen2.5-1.5b)")

client = OpenAI(base_url="http://localhost:11434/v1", api_key="ollama")

def refine_prompt(user_input):
    system_instruction = """
    You are a Prompt Engineering expert. 
    Your goal is to transform messy Chinese ideas into high-quality, precise, and structured ENGLISH prompts.

    Format: 
    - [Role]: ...
    - [Task]: ...
    - [Constraints]: ...
    
    IMPORTANT: You must output the full optimized prompt. 
    At the end of [Constraints], always add: "Provide the response concisely, avoid unnecessary explanations, and output only the core result."
    """
    
    response = client.chat.completions.create(
        model="qwen2.5:1.5b",
        messages=[
            {"role": "system", "content": system_instruction},
            {"role": "user", "content": f"Optimize this requirement: {user_input}"}
        ],
        temperature=0.3,
    )
    return response.choices[0].message.content

# UI 介面設計
st.title("Prompt Assistant (Qwen2.5-1.5B)")
st.markdown("---")

col1, col2 = st.columns(2)

with col1:
    st.subheader("請輸入")
    user_query = st.text_area("輸入內容：", height=200)
    input_len = len(user_query)
    st.caption(f"原始字數：{input_len}")

if st.button("優化 Prompt"):
    if user_query:
        with st.spinner("優化中..."):
            start_time = time.time()
            refined = refine_prompt(user_query)
            end_time = time.time()
            
            with col2:
                st.subheader("優化後的 Prompt")
                st.success(refined)
                output_len = len(refined)
                st.caption(f"優化後字數：{output_len}")
            
            st.markdown("---")
            c1, c2, c3 = st.columns(3)
            c1.metric("耗時", f"{end_time - start_time:.2f}s")
            c2.metric("資訊增量", f"{(output_len/input_len if input_len > 0 else 0):.1f}x")
            c3.metric("模型", "Qwen2.5-1.5B")
            
            st.code(refined, language="markdown") 
    else:
        st.warning("請輸入內容")
