Face Attendance System

一套 Python 桌面出勤系統：使用 MediaPipe 偵測人臉、InsightFace 產生 512 維臉部特徵、FaceMesh 進行眨眼活體檢測，最後把員工資料與出勤紀錄寫入 SQLite。

PySide6 GUI MediaPipe InsightFace SQLite OpenCV

Mode 01

註冊

擷取單一人臉，抽取特徵向量，將員工 ID、姓名與 embedding 存入 SQLite。

Mode 02

打卡

辨識員工身分後記錄上班、下班、休息等出勤狀態，並用冷卻時間避免重複打卡。

Mode 03

查詢

依日期範圍查詢出勤紀錄，顯示信心分數、時間戳記與打卡類型。
